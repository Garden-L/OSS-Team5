### 오픈소스 SW 개론 Team5 _ Tic Tac Toe
=======================================

c언어로 구현한 틱택토 게임입니다.

### 설치방법
Git bash 에서 다음과 같은 명령어를 입력합니다.
```
$ git clone https://github.com/JeongWonLim/OSS-Team5 downloadTTT
```

### 개발환경
```
운영체제 : Windows 10
개발도구 : Visual Studio 2015
```
